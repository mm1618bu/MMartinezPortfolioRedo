# Warehouse Employee Manager

A React application for managing warehouse employees, staffing levels, and time-off requests.

## Features

- Monitor staffing levels by department and shift
- Add Voluntary Extra Time (VET) for employees
- Manage Unpaid Time Off (UTO) requests
- Handle Voluntary Time Off (VTO) requests
- Approve/deny PTO (Paid Time Off) requests

## Getting Started

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

The app will run on [http://localhost:3000](http://localhost:3000)

## Usage

Navigate through the dashboard to:
- View current staffing levels
- Add time-off requests for employees
- Approve or deny PTO requests
